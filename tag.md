设计一个文章网站的 UI 界面，核心在于“内容为王”。我们需要打造一个既美观又极具可读性的界面，让读者能够沉浸在文字中。

下面我为你整理了一套现代、极简且功能完备的文章网站 UI 设计方案：

🎨 一、 整体视觉风格与规范

设计风格：采用极简主义（Minimalism），强调留白（Negative Space），减少视觉干扰，突出排版和内容本身。
色彩搭配：
    主色调：黑白灰为主（例如背景 #FAFAFA 或纯白 #FFFFFF，正文文字 #333333）。
    强调色：选择一种低饱和度的颜色作为点缀（如克莱因蓝、墨绿或珊瑚橙），用于按钮、链接和标签，引导用户视线。
字体排印：
    标题：使用衬线体（Serif，如 Noto Serif SC），营造优雅、文学的气质。
    正文：使用无衬线体（Sans-serif，如 Inter, Roboto, 思源黑体），保证屏幕阅读的清晰度。
    行高：正文行高建议设置为字号的 1.6-1.8 倍，提升阅读舒适度。

🖥️ 二、 核心页面布局设计

首页 / 列表页 (Homepage)
这是用户的第一印象，需要快速展示最新、最热门的内容。

顶部导航栏 (Sticky Header)：
    左侧：品牌 Logo。
    中间：主要分类（如：科技、生活、观点、教程）。
    右侧：搜索图标（点击展开搜索框）、夜间模式切换开关、登录/订阅按钮。
首屏推荐区 (Hero Section)：
    采用 1+2 布局：左侧一张大图展示头条文章（带标题摘要），右侧两篇次级推荐文章（小图+标题）。
文章瀑布流/网格 (Content Grid)：
    卡片式设计。每个卡片包含：特色图片 -> 分类标签 -> 文章标题 -> 简短摘要 -> 底部元数据（作者头像、发布时间、预计阅读时间）。
    交互：鼠标悬停时，卡片轻微上浮并出现阴影。
侧边栏 (Sidebar - 可选)：
    “关于作者”简介。
    “热门标签”云。
    “邮件订阅”输入框。

文章详情页 (Article Page)
这是网站的核心，重点在于沉浸式阅读体验。

布局结构：单栏居中布局（宽度控制在 700px - 800px 之间，避免眼球移动幅度过大）。
文章头部：
    面包屑导航（首页 > 分类 > 文章）。
    H1 大标题（醒目）。
    元信息栏：作者、发布日期、最后更新时间、浏览量。
正文区域：
    首字下沉（可选）：增加杂志感。
    引用块：使用不同的背景色或左侧竖线样式，突出金句。
    图片/图表：全宽或居中对齐，下方附带灰色小字注释。
    代码块：如果是技术文章，需提供语法高亮和一键复制按钮。
辅助功能悬浮栏：
    在文章左侧或右侧设置悬浮工具条：包含“分享”、“收藏”、“评论”以及“返回顶部”。
文章底部：
    文章标签。
    作者简介卡片（带头像和关注按钮）。
    “上一篇 / 下一篇”导航。
    相关推荐文章（3篇）。

📱 三、 移动端适配 (Responsive Design)

汉堡菜单：将导航栏折叠进左上角或右上角的汉堡图标中。
单列布局：首页的网格布局自动转换为单列垂直滚动。
触控优化：按钮和链接的点击区域至少为 44x44 像素。
底部操作栏：在文章详情页，可以将“评论”和“目录”固定在屏幕底部，方便单手操作。

🛠️ 四、 交互与微动效

加载状态：使用骨架屏代替传统的旋转 Loading 图标，减少用户等待焦虑。
阅读进度条：在页面顶部显示一条细长的进度条，随滚动逐渐填满，告知用户剩余阅读量。
图片懒加载：图片进入视口时才加载，并带有淡入效果。
